# 🎓 RappterAcademy

> *Learn AI by raising one.*

RappterAcademy is the educational on-ramp for RAPP Brainstem. It's not a course — it's an interactive journey from "what is AI?" to "I built my own." Every lesson follows the same principle as the platform itself: **the story is the architecture.**

Whether you're 8 or 80, you start the same way: you hatch a rappter and watch it come alive.

## The Philosophy

This is modeled on how humans actually learn — not how software docs are usually written.

- **Khan Academy** taught us: meet people where they are, let them go at their own pace
- **Montessori** taught us: learn by doing, not by reading about doing
- **The biology metaphor** teaches us: if you can understand a living thing, you can understand this

Every lesson has:
- 📖 **A story** — what's happening and why it matters
- 🔬 **A try-it** — something to run on your own machine
- 💡 **An insight** — the deeper principle underneath
- 🪜 **A next step** — where to go from here

The journey from student to master is never finished. But the on-ramp should be wide, gentle, and impossible to fall off of.

## For Educators

RappterAcademy is designed to be linked, embedded, and adapted. Each lesson is a standalone HTML page. Use them in:

- Classroom settings (CS101, intro to AI, STEM programs)
- Workshop demos (hackathons, meetups, conferences)
- Self-guided learning (send someone a link, they're off)
- Parenting moments ("want to see something cool?")

The content is open source. Fork it, translate it, remix it.
